## [1.0.1] - 2026-04-14

### 점수 설정 UX 개선
- 점수 설정 2탭 → **1탭 통합** (세트별 좌 | 메인옵별 우)
- 등급 구간 판별: 정확 일치 → **구간 기반** (1.5 같은 중간값 처리)
- 기대값 표기 통일: 강초(망각) / 옵초(혼돈), 현재(최저) / 평균 기대 / 최대 기대
- 태생 4옵 보라색 표시 (주요 옵션인 경우)
- 점수 초기화 대화상자 추가, 미저장 보호 (탭 변경 / 종료 시 경고)

### 기타
